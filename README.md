# lukemjoyce

Personal site for Luke M. Joyce — a sleek, minimal home page.

- Monospace type (JetBrains Mono), white background, stone-colored text
- Hamburger menu (top left) that smoothly slides out a navigation panel:
  **Luke M. Joyce · About · Hobbies · Experience**

## Run

It's a static site — just open `index.html`, or serve the folder:

```sh
python -m http.server 8000
# then visit http://localhost:8000
```

## Files

- `index.html` — markup
- `styles.css` — styling and transitions
- `script.js` — menu open/close behavior
